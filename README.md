# arp_response

## Scripts

Some tcl scripts were added to synthesize and simulate the module using the Vivado toolset. This was tested with Vivado 2021.1.

### Steps to Synthesize

1. Open Vivado
2. `cd {arp_response_repo}/scripts`
3. `source ./synth.tcl`

### Steps to Simulate

1. Open Vivado
2. `cd {arp_response_repo}/scripts`
3. `source ./synth.tcl`
